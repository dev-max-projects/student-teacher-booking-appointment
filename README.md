# 📚 Student Teacher Appointment Booking App

A simple web application built using **HTML, CSS, and JavaScript** that allows students to book appointments with teachers.  
The app helps manage appointment requests in an easy and organized way.

## Features
- Book appointment with teacher
- Enter student details
- Select date and time
- View booked appointments
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project demonstrates basic **form handling and front-end functionality** using JavaScript.

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
